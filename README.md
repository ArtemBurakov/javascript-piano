# javascript-piano

Piano tiles game written in Javascript, HTML and CSS.

# Dependencies

This game does not require any dependencies.

# Installation

Extract all files from downloaded folder. Run `index.html` file.

# How to play:

All you need is to tap on the black tiles as they appear from the top of the field.
